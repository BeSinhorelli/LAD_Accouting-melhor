# LAD Accounting Dashboard

## Executar no GitHub Codespaces
1. Abra a aba de Codespaces no repositório do [LAD Accounting Dashboard](https://github.com/LAD-PUCRS/LAD-Accounting-Dashboard)
![Codespace](assets/screen.png)

2. Você será redirecionado no terminal de um VS Code virtual

3. Após carregar todo o script (automático), uma conexão será estabelecida no terminal do VS Code virtual
4. No pop-up, clique em `abrir no browser`

## Executar em servidor local

### Requisitos

- Python 3
- Dash 2.0.0 - `pip install dash`
- Pandas - `pip install pandas`
- dash-bootstrap-components - `pip install dash-bootstrap-components`
- openpyxl - `pip install openpyxl`

### Executar

Após instalar as bibliotecas, acesse o diretório com as pastas/arquivos deste repositório na máquina local, digite o comando abaixo e aguarde o link para visualizar o dashboard:

`python lad.py`

## Executar no jupyter-lab

### Requisitos

- python 3
- jupyter-lab
- jupyter-dash `pip install jupyter-dash` 

Dentro do diretório com as pastas/arquivos deste repositório em sua máquina local, digite o comando para acessar o jupyterlab:

`jupyter-lab`

- Acesse o arquivo lad.ipynb e execute as células para instalar as bibliotecas. 
- Execute a célula com o código e aguarde o link para visualizar o dashboard.
